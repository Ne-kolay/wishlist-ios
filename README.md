# Wishlist iOS App

This is an iOS application for managing a user's wishlist. Users can create and manage items they want to buy or receive, directly from their iPhone.

## Features

- Basic navigation
- Registration and login/logout

**Note**: User interaction features (such as sharing wishlists, following other users, etc.) are still under development and will be added in future releases.

## Technologies Used

- Swift
- UIKit 

## How to Run the Project

1. Clone the repository:
    ```bash
    git clone https://github.com/Ne-kolay/wishlist-ios.git
    ```

2. Open the project in Xcode:
    ```bash
    open Wishlist.xcworkspace
    ```

3. Build and run the project in the simulator or on a connected device.

4. Access the app on your device or simulator.
